Setup in 30 Seconds

Install any server, add it to your Claude Code config, and you're done.

1

Install

pip install mcp-dns-intel

2

Configure

Add to claude_desktop_config.json or .claude.json

3

Use

Claude can now call whois_lookup("github.com") directly

claude_desktop_config.json
{ "mcpServers": { "dns-intel": { "command": "python", "args": ["-m", "src.server"], "cwd": "/path/to/mcp-dns-intel" } } }
